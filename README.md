# Copa Mundial Femenina FIFA 1991–2023

Análisis histórico de rendimiento de selecciones en la Copa Mundial Femenina.

## Datasets
- `world_cup_women.csv` — Información general por edición del torneo
- `matches_1991_2023.csv` — Detalle de los 348 partidos (1991–2023)

## Contenido del análisis
1. Análisis exploratorio de variables, nulos y duplicados
2. Validación cruzada entre tablas
3. Tabla de posiciones — Mundial 1991
4. Tabla de goleadoras — Mundial 2023
5. Tabla resumen consolidada (todas las ediciones y equipos)

## Cómo ejecutar
```bash
pip install pandas numpy
jupyter notebook Copa_Femenina.ipynb
```
